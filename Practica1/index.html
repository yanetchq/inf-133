1. Layout Estático

Un layout estático es aquel en el que los elementos tienen un tamaño fijo y no cambian ni se ajustan según el tamaño de la pantalla o la ventana del navegador. Es decir, todo el contenido tiene un ancho y altura fijos (en píxeles), y no se adapta a pantallas más grandes o más pequeñas.

2. Layout Fluido (Liquid Layout)

Un layout líquido es un diseño web en el que los elementos de la página se ajustan de manera dinámica al ancho de la ventana del navegador. A diferencia de un layout estático, donde los elementos tienen tamaños fijos, un layout líquido utiliza unidades relativas como porcentajes en lugar de píxeles, permitiendo que los contenidos se adapten al tamaño de la pantalla, tanto en dispositivos grandes como pequeños.

3. Layout Responsivo

Un layout responsivo es un diseño web que se adapta de manera flexible a diferentes tamaños de pantalla (por ejemplo, desde escritorios grandes hasta dispositivos móviles). Utiliza unidades relativas (como porcentajes, em, rem, o vw) y las media queries de CSS para ajustar el diseño según las características del dispositivo (ancho, altura, resolución).

4. Layout Adaptativo (Adaptive Layout)

Un layout adaptivo es un diseño web que se ajusta a varios tamaños de pantalla utilizando distintos diseños predeterminados para rangos de dispositivos específicos (como escritorio, tabletas y móviles). A diferencia de un layout responsivo, que ajusta fluidamente los elementos según el tamaño de la pantalla, un layout adaptivo utiliza diseños fijos para varios anchos específicos de pantalla y se adapta según el dispositivo o resolución.
Un layout adaptivo puede ser útil cuando se necesita más control sobre cómo se presentan los elementos en diferentes tipos de dispositivos, aplicando un conjunto de reglas de estilo específicas para cada tipo de pantalla.

El CSS Grid Layout es un sistema de diseño bidimensional que te permite crear diseños complejos utilizando filas y columnas. 6. Layout Basado en Flexbox

El Flexbox Layout es un sistema de diseño unidimensional que permite distribuir el espacio entre los elementos de un contenedor, alinearlos y distribuirlos de manera flexible. Es ideal para crear layouts donde el tamaño de los elementos puede variar, y se adapta bien a pantallas de diferentes tamaños.
